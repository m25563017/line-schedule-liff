# LINE 共編排程

以 Vue 3、Vite、LINE LIFF 與 Firebase 打造的 LINE 群組共編排程應用。在 LINE 內建立群組、發起活動，成員在月曆上勾選有空的日子，主揪可一鍵定案日期，減少來回喬時間的訊息。

---

## 專案簡介

**LINE 共編排程** 針對「約時間」情境設計：主揪在 LINE 建立群組、預先建立虛擬成員名額，再發起活動並設定可選月份；成員透過邀請連結加入並認領身分後，在月曆上點選有空的日期。系統會統計最多人有空的 Top 3 日期，主揪可一鍵決定日期定案；定案後成員無法再修改空檔，避免反覆更改。

- **群組**：可設定名稱、封面圖與成員（含虛擬成員，供之後認領）。
- **活動**：每個群組可有多個活動，每個活動有標題與可選月份範圍（最多 3 個月），成員在月曆上勾選空檔。
- **主揪**：可代填虛擬成員空檔、決定活動日期；定案前會提醒尚有成員未填寫。不需將機器人加入 LINE 群組；有人未填寫時，主揪可「發送 LINE 提醒」；決定日期時會自動對所有成員發送定案通知（皆為一對一推播，免付費 Push）。
- **邀請**：產生專屬加入連結，可複製或一鍵分享 LINE 邀請卡片。
- **月曆**：以月為單位切換、顯示臺灣假日，點選日期可查看該日誰有空。

使用方式：在 LINE 聊天室對機器人傳送訊息或加好友時，機器人會回傳「打開排程」按鈕，點擊後開啟 LIFF 頁面。建立群組與發起活動均在 LIFF 網頁內完成，機器人不需加入任何 LINE 群組。

適用於社團、讀書會、球聚、出遊等需要多人一起喬時間的場合。

## 功能一覽

| 類別 | 功能                                                                                       |
| ---- | ------------------------------------------------------------------------------------------ |
| 登入 | LINE LIFF 登入、開發模式模擬帳號                                                           |
| 群組 | 建立、列表、詳情、編輯（名稱／封面／成員）、刪除、邀請連結、LINE 分享卡片                  |
| 成員 | 虛擬成員預先建立、邀請連結加入並認領身分、權限（admin／editor／viewer）、移除成員          |
| 活動 | 發起活動（標題與月份範圍）、活動詳情、編輯（標題／月份／清除資料）、刪除活動               |
| 排程 | 月曆選日期填寫空檔、主揪代填、查看每日可參加名單、Top 3 推薦日期、決定日期定案、定案後鎖定 |
| 提醒 | 未填寫時主揪可發送 LINE 提醒；決定日期時自動對所有成員發送定案 Flex（一對一，免 Push）     |
| 其他 | 台灣假日標示、封面圖上傳（Firebase Storage）、錯誤與載入狀態處理                           |

## 技術棧

- **前端**：Vue 3、Vue Router、Vite、Tailwind CSS、LINE LIFF、@pieda/core（通知）
- **後端與資料**：Firebase（Firestore、Storage、Functions）、LINE Messaging API

## 安裝與本地開發

```bash
# 安裝依賴套件
npm install

# 啟動開發伺服器
npm run dev

# 建置正式版
npm run build

# 預覽建置結果
npm run preview
```

開發模式下會使用模擬帳號，無需 LINE 登入即可操作介面。

---

## 後端（LINE Webhook）

後端程式位於 `functions` 目錄：

```bash
cd functions
npm install
```

請在 `functions` 目錄下設定 `.env`，填入 `CHANNEL_ACCESS_TOKEN`、`CHANNEL_SECRET`；若需覆寫 LIFF ID 可設定 `LIFF_ID`。部署指令：

```bash
firebase deploy --only functions
```

提醒與定案通知會呼叫 `remindPendingUsers`、`notifyEventFinalized`。若 Functions 部署於其他 region，可設定前端環境變數 `VITE_FUNCTIONS_BASE` 指向正確網址。

## 權限說明

- **admin**：群組建立者。可編輯群組、管理成員、刪除群組，並可編輯／刪除任何活動、代填虛擬成員空檔、對任何活動定案。
- **editor**：可編輯成員。可發起活動，對「自己建立」的活動擁有完整管理權（編輯、刪除、定案），並可代填群組內虛擬成員空檔；對他人建立的活動僅能填寫或修改自己的空檔。
- **viewer**：僅可查看。只能查看群組與活動，並填寫／修改自己的空檔，無法建立或編輯活動，也不能定案。

活動的編輯、刪除與定案權限僅限於該活動的建立者（`createdBy`）與群組 admin。

---

## 已知限制與可加強項目

- **Webhook**：請求未實作 LINE 簽章驗證，建議補上以提升安全性。
- **活動內容**：目前僅有「日期」，無具體時間、地點或備註欄位；可考慮在活動詳情或編輯頁擴充。
- **通知**：本服務為免費使用，不實作付費 LINE Push 訂閱；無活動日前一天提醒等排程提醒。
- **體驗**：網路或 Firestore 錯誤時尚無明確重試或離線暫存機制；無群組／活動搜尋與深色模式；無法匯出空檔統計或定案結果為圖片或連結。
